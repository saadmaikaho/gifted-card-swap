# voucher-swap
App designed for a seamless and secure way for users to swap gift vouchers, gift cards and gift certificates of various types.

	<img src="https://skillicons.dev/icons?i=vue,js,html,css,sass,bun,vite" />

## Project Setup

### Installation

```sh
npm install -g bun
bun install
```

### Development build

```sh
bun dev
```

### Production build

```sh
bun build
```
